# Quiz Application Microservice

This repository contains the source code for the Quiz Application, a microservice-based project.

## Architecture

The development of this application will follow a phased approach. We will begin by building it as a monolithic application. This initial version will contain all the core features and functionalities within a single, self-contained unit.

Once the monolithic application is stable and functional, we will undertake the process of re-architecting it into a microservices-based architecture. This will involve breaking down the application into smaller, independent services, each responsible for a specific business capability. This transition will allow for greater scalability, flexibility, and maintainability.
